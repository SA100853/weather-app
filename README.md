# 🌦️ Weather Forecast App

A responsive web application built with **Flask** and **OpenWeatherMap API** that provides a **5-day weather forecast** for any city using embedded **Geocoding** for accurate location data. Includes a **light/dark mode toggle**, sunrise/sunset times, humidity, and wind speed. Deployed on **Render**.

## 📌 Features

- 🔍 City-based weather lookup using Geocoding API
- 📅 5-day forecast with weather icons and dates
- 🌄 Displays sunrise and sunset times
- 💧 Shows temperature, humidity, and wind speed
- 🌓 Dark and Light theme toggle
- ❤️ Footer credit: "Made with ♥️ by Sarfaraj"
- 🚀 Deployed on Render
- 📱 Fully responsive design (mobile + desktop)
